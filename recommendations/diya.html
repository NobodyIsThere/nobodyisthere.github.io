Games for Diya

So you’re a certified PC GAMER now. As promised, here’s a list of personalised recommendations for you.

Multiplayer

A Hat in Time

Do you like Mario? I don’t, but I enjoyed playing A Hat in Time with James. It’s cute and it has a good photo mode.

Magicka

Freely combine elements to make spells that accidentally kill your friends. Very silly, very fun. Also very hard towards the end, but you can probably pick it up for about 75p so I’d say it’s worth it even if you don’t finish it.

Content Warning

Lovely Lethal Company–like extraction horror job where you make videos with your friends.

If you like this, then you might also like R.E.P.O.

Single-player

Danganronpa

You know you should play Danganronpa.

Phoenix Wright: Ace Attorney

Obviously.

80 Days

Interactive fiction adventure game based on Jules Verne’s Around the World in 80 Days. Beware the utterly pointless real-time element. I think it might have been added for the PC release, so consider getting the mobile version for the best experience.

Crypt of the NecroDancer

Rhythm dungeon crawler with one of the best soundtracks you’ve ever heard. Oh, and Hatsune Miku is in it. She might be DLC, though? Not sure. However much they charge for this game, it isn’t enough.

If you like this, then you might also like Cadence of Hyrule: Crypt of the NecroDancer feat. The Legend of Zelda

Disco Elysium

The only video game RPG I’ve ever played that actually feels like an RPG.

Papers, Please

The quintessential indie darling.

Stray Gods

It claims to be an RPG, but it’s more of a visual novel. But it’s a musical and it features Greek mythology, so I thought you’d be into it. I was slightly disappointed, but maybe my expectations were too high. It’s definitely worth a look.

Necrobarista

Lovely atmosphere.

If you like this, then you might also like VA-11 Hall-A (which I haven’t played yet but it’s supposed to be amazing)

Sayonara Wild Hearts

Stylish rhythm-action album.

Braid

Why are there so many puzzle games on this list? Because they’re all really good.

Firewatch

Great if you want to feel like you’re outside without having to go outside.

If you like this, then you might also like Gone Home, Journey

Outer Wilds

Yeah, technically it’s a puzzle game, and the controls can be a bit tricky, but there is no way I’m leaving this one off the list. If you liked Return of the Obra Dinn, you’ll probably like Outer Wilds.

Heaven’s Vault

You play as an archaeologist/linguist piecing together fragments of an ancient language. The way you choose to interpret the past shapes how you think about the present.

Hades

People love it, and it has sexy characters from Greek mythology.

Portal

It’s a classic, and the game that got me into gaming.

If you like this, then you might also like Portal 2. Obviously.

The Walking Dead

Telltale’s version. Telltale really set up the episodic “choices matter” genre with this game. It’s good!

If you like this, then you might also like Life Is Strange

Emily Is Away

Another that I haven’t got around to yet. Chatroom-based story game.

The Room

V’s company made it! V didn’t work on it, though. It’s an escape room/puzzle box game. So like Zero Escape but without the visual novel parts. So like Zero Escape but without the good parts. It’s good though.

Celeste

I haven’t played Celeste, but I’ve been meaning to for a while. It’s supposed to be a tough but rewarding platformer that’ll bring a tear to your eye.

If you like this, then you might also like Super Meat Boy, N
